# Daniyel's Portfolio

## [Project 1: beauty salon](https://docs.google.com/spreadsheets/d/14nQ7rq9XkTimU7ro1P5MD5drHhpQ_UciFkwg41hRIQk/edit?usp=sharing)
Анализ данных о заказах барбершопа за январь–июль 2026 года (5 356 транзакций), выгруженных из CRM-системы. Цель — оценить финансовые результаты бизнеса и определить, какие маркетинговые каналы (промокоды) реально окупаются, а какие только съедают маржу.


* Собрана и очищена выгрузка операций: доходы (услуги/товары) и расходы (зарплата, реклама, аренда, закупки)
* Построены сводные показатели по месяцам, категориям услуг и способам оплаты
* Рассчитан ROI для 7 маркетинговых каналов на основе суммы предоставленных скидок и выручки, которую они принесли
* Визуализированы ключевые метрики для презентации результатов

### Финансовая картина
За период выручка составила 16 428 295 ₽, расходы — 9 243 900 ₽, операционная прибыль — 7 184 395 ₽. Данные за июль охватывают только первую неделю месяца, поэтому итог за июль ниже остальных не из-за спада, а из-за неполного периода.

### Эффективность маркетинговых каналов

### Выводы
* Органический поиск (SEO/контекст) — самый рентабельный канал. «Поисковые запросы в Яндексе» дают ROI 19x при небольшой скидке 5% — почти бесплатный трафик, который стоит усиливать (SEO, локальная выдача, отзывы на картах).
* Посты в Telegram убыточны с точки зрения скидки. Скидка 50% съедает всю дополнительную выручку (ROI 1,0x) — по сути, клиенты получают услугу по себестоимости. Либо снизить размер скидки, либо пересмотреть формат акции.
* VK-каналы (посты, сторис, реклама) — середняки с ROI 4–6x: работают, но проигрывают поиску и «обещанным скидкам». Их стоит сузить по таргетингу, а не сокращать полностью.
* Основная точка роста прибыли — не реклама, а ФОТ. Зарплата — 75% расходов против 10% на рекламу, поэтому даже удвоение рекламного бюджета не изменит экономику так сильно, как оптимизация загрузки мастеров или пересмотр схемы оплаты труда.
* Ядро выручки — услуги, а не товары (97% против 3%), значит, upsell сопутствующих товаров (шампуни, гели) — недоиспользованный резерв: только 545 из 5356 транзакций включали товар.
* Данные за июль неполные (только 1 неделя) — при обновлении дашборда логично исключать текущий месяц из трендовых сравнений, чтобы не искажать динамику.
